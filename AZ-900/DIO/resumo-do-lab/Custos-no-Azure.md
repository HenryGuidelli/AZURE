# Otimizando Custos no Azure

Os custos de operar no Azure variam conforme a atividade e frequência de uso, mas há maneiras de estimar e otimizar os custos no Azure.

### Estimando
Para estimar custos no Azure pode-se usar a Calculadora de Preços e Calculadora TCO.
Nesta fase, estabelece-se a base lógica para o recebimento dos recursos:
- ***Calculadora de Preços*** - Fornece um custo estimado para provisionamento de recursos no Azure.
- ***Calculadora TCO*** - Fornece estimativas comparadas entre o custo ao longo do tempo da Azure e um datacenter tradicional.
Há muitas formas de utilizar essas ferramentas para obter a estimativa mais ***próxima*** da realidade possivel.

### Otimizando
A Azure oferece recursos e ferramentas que ajudam a otimizar os custos, entre elas:
- ***Gerenciamento de Custos*** - Fornece a capacidade de verificar rapidamente os custos de recursos do Azure, criar alertas com base nos gastos com recursos e criar orçamentos.
- ***TAGs*** - Fornecem informações adicionais, ou metadados, a respeito de seus recursos.
- ***Balanceamento de Carga*** - Configuração de balanceadores públicos ou internos para distribuição de tráfego entre instâncias de computação.

---

## Considerações sobre Otimização de Custos
A gestão de custos da infraestrutura envolve o uso do **Azure Cost Management** para visualização geral e rápida nocontrole de gastos e uso de outros recursos como as **TAGs** para otimização.
